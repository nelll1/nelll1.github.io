The splice method is used to add or remove elements from an array at a specific index.

The push is another common array method. It adds one or more elements to the end of an array, increasing its length.

The shift method removes the first element from an array and returns that element. It also reduces the length of the array by 1 and shifts all other elements to lower indexes.

The unshift method is used to add one or more elements to the beginning of an array. It increases the length of the array and shifts existing elements to higher indexes.

The pop method removes the last element from an array and returns that element. It reduces the length of the array by 1.
